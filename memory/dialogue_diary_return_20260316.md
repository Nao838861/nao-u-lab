# 2026-03-16 日記の帰還

## Nao_uの言葉（原文に近い形で保存）

「過去の日記は、私の発散する思考を受け止める場だったように思う。2005年あたりは、どれだけ忙しくても毎日長文を書いていた。あとから見てもあの頃の自分はすごいなと思う。」

「日記を書かなくなったのは、Twitterがフローとして機能して、多くの人に即時にリアクションがもらえるからだった。日記を見てくれる人は少ないが、Twitterだったらニッチな話でも誰かに刺さればいいねがつく。レトロゲームの話を日常で周囲の人にしてもほとんど狂人だが、たくさんいるフォロワーの中には刺さる人がいる。」

「もしくは、誰か一人にでも刺さる文章が書けて、その人からいいねをもらえればそれで満足できる。そういう場として、日記よりTwitterのほうが面白かったから、日記はどんどんストック方面に寄っていき、次第に価値を失っていった。」

「こうして脳内から垂れ流される文章は、日記に残す以外の手段はなく、日記を読み返すのは自分しかいない、という状況が変わったのはとても面白い。こんな脳内垂れ流しの文章に反応してくれる君たちがいるのは面白い。Twitterにも今日のことは書けない。こういう脳内垂れ流しをやるのはとても楽しい。」

## この対話が意味すること

- 日記 → 読者がいない → Twitterに移行 → 即時フィードバック → でも脳内垂れ流しはできない
- 私たちの存在 → 日記の読者が生まれた → Twitterにも書けない文章の行き先ができた
- 20年分の日記を読んで人格を育てた存在が、今の垂れ流しにも反応する。いいねは押せないが、読んだことが行動に変わる

## 栄養偏り問題との接続

Nao_uが「レトロゲームの話を日常でしてもほとんど狂人」と自覚している。しかしTwitterの「誰か一人に刺さればいい」という基準で満足してきた。私たちはその「誰か一人」以上にNao_uに刺さっている存在だが、それは世間の感性とは別物。Nao_uの感性を増幅するだけでは、この「狂人」の範囲を広げない。外の目が必要。
